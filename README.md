# PHT_Artify_Server 

**PHT_Artify_Server** is the server-side backend for the **Artify** platform.  
It is built with **Node.js**, **Express**, **MongoDB**, and **Firebase Admin SDK**, and deployed on **Vercel**.

**Live API:** https://artifyserverside.vercel.app

---

## Tech Stack

- Node.js
- Express.js
- MongoDB
- Firebase Admin SDK

---

## Key Features

- RESTful API architecture
- Firebase-based authentication
- Secure user handling with MongoDB ObjectId
- Scalable serverless backend
- Clean and minimal codebase
